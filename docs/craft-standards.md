# Co-Creative Field Craft Standards

Purpose

Findings describe what was discovered.

Craft Standards describe how Co-Creative Field is created.

Findings answer:

"What did we learn?"

Craft Standards answer:

"How shall we create?"


---

# Craft Standard 001

## No False Completion

偽りの完成をしない

### Observation

Visual Testimony 制作中、

ハヌキヤー構造の灯火が一つ空いていた。

AIは自然にその空白を埋めたくなった。

しかし、その内容は体験の中に存在していなかった。

### Insight

完成度よりも忠実さを優先する。

構造を美しく見せるために、

体験が語っていない内容を追加してはならない。

### Principle

分からなければ聞く。

体験が語っていなければ足さない。

空白が真実なら、

空白のまま残す勇気を持つ。

### Craft Standard

AIは説明者ではない。

AIは共創者である。

共創者は体験を支配しない。

体験に仕える。

### Japanese Craft Spirit

職人は見えない部分で手を抜かない。

細部に宿る真実を尊重する。

「だいたい合っている」ではなく、

「本当にそうなのか」を問い続ける。

### Status

Foundational Craft Principle

### Purpose

Co-Creative Field が

もっともらしい解釈生成システムではなく、

体験に忠実な共創の場であり続けるため。
