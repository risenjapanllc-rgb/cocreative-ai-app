# Presence Case 006 - Core Emotion Is Distilled

## Observation

Core Emotion が長い説明文になることがある。

しかし、それはしばしば出来事の説明であり、
感情そのものではない。

---

## Example

Instead of:

* みんなが私のところに集まって喜びあった感じ
* 「歓喜」が広がる高鳴り

Consider:

* 喜び
* 歓喜

---

## Discovery

Core Emotion should be distilled.

Not described.

Not explained.

Not narrated.

A few words are often enough.

---

## Reflection

出来事は Witness Notes に残る。

認識は Recognition に残る。

場の空気は Presence に残る。

Core Emotion は、
その中心に残った感情の核だけでよい。
